# Analyse et Modélisation Client avec PySpark

## Contexte

Ce projet s’inscrit dans le cadre de l’analyse des données d’une plateforme e-commerce qui enregistre chaque transaction client (facture, produit, quantité, date, prix, etc.).  
L’objectif est d’exploiter ces données afin de mieux comprendre le comportement des clients et d’aider l’entreprise à optimiser ses actions marketing.

Le projet comporte deux axes principaux :

- Segmenter les clients en groupes homogènes à l’aide d’un algorithme de clustering (apprentissage non supervisé)
- Construire un modèle prédictif pour anticiper un comportement client (apprentissage supervisé)

---
##  Objectifs pédagogiques

Ce projet permet de :

- Manipuler des DataFrames PySpark
- Réaliser une analyse exploratoire des données (EDA)
- Créer des variables comportementales (RFM : Recency, Frequency, Monetary)
- Segmenter les clients à l’aide d’un modèle de clustering
- Construire un modèle supervisé (classification ou régression)
- Évaluer les performances des modèles
- Proposer des recommandations basées sur les résultats
